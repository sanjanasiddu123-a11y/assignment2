# Authentication System Interface

This repository contains a simple **Authentication System User Interface** built using **HTML, CSS, and Bootstrap 5**.
The project demonstrates the structure and design of a multi-page authentication workflow.

The application consists of several pages that simulate the process of user login, account registration, password recovery, and accessing a dashboard.

---

## Project Pages

The following pages are included in this project:

* Login Page
* Registration Page
* Forgot Password Page
* Reset Password Page
* Dashboard Page

Each page is styled using Bootstrap components and custom CSS for a clean layout.

## How the Application Works

1. The user opens the **Login Page** and enters credentials.
2. If the user does not have an account, they can navigate to the **Registration Page**.
3. If the user forgets their password, they can use the **Forgot Password Page**.
4. The password can be updated through the **Reset Password Page**.
5. After login, the user is redirected to the **Dashboard Page**.
