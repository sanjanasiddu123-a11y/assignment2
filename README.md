# 🔐 Authentication System UI

This repository contains a **simple authentication interface** developed using **HTML, CSS, and Bootstrap 5**.
The project demonstrates the user authentication flow including login, registration, password recovery, and dashboard interface.

The aim of this project is to create a **clean and responsive authentication layout** using Bootstrap components.

---

## 📌 Project Overview

The project includes several pages that simulate a basic authentication system:

* Login Interface
* User Registration Form
* Forgot Password Page
* Reset Password Page
* Dashboard Page

Each page is styled using **Bootstrap components and custom CSS** to create a responsive and user-friendly interface.

---

## 🛠 Technologies Used

The project is built using the following technologies:

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

## 📖 Application Flow

1. Users enter credentials on the **Login Page**.
2. If the user does not have an account, they can register through the **Registration Page**.
3. Users who forget their password can access the **Forgot Password Page**.
4. Password can be updated through the **Reset Password Page**.
5. After login, users are redirected to the **Dashboard Page**.
